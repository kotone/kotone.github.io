部署流程

设置Menu

  1. 首先创建一个目标页面（举例创建一个tags页面）

  2. 根目录source 就会多出一个tags文件夹，接下来修改tags文件夹下的index.md

  3. 修改主题下的_config.yml 的menu 选项，增加tags 选项 || 后面是menu 的 icon

  4. 修改主题下的 languages/zh-Hans.yml

修改背景

找到主题下的 next\source\css_common\outline\outline.styl 修改body的样式就可以了
